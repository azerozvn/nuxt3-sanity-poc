# Nuxt3 Sanity Demo

## Development

```bash
# Install dependencies
yarn install

# Start app dev
cd nuxt-app && yarn dev

# Start CMS studio
cd sanity && yarn dev
```

- [http://localhost:3000](http://localhost:3000) - App
- [http://localhost:3000/admin](http://localhost:3333) - CMS
